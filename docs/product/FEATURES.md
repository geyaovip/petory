# Petory 功能说明（V1.4）

## 桌宠与桌面

- 多宠并行：免费 1 只 / Pro 5 只同时显示在桌面
- 主宠：聊天、成长、番茄钟状态绑定主宠
- 同伴宠：可显示在桌面，点击可短暂切换开心姿势（有多姿势图时）
- 点击穿透：默认开启，透明区域不挡桌面操作；**无关闭开关**
- Alpha 命中：根据 PNG 透明通道，仅宠物本体可点击/拖拽

## 生成与姿势

- 免费 3 种姿势：待机、开心、提醒
- Pro 6 种姿势：+ 专注、睡觉、生气
- Pro 姿势补全：升级后为已有宠物补生成缺失姿势（不扣额度）
- 单姿势重生成：宠物管理 → 对已拥有的姿势单独重生成（不扣额度）

## 陪伴与成长

- AI 聊天（Kimi），按性格生成气泡回复
- 番茄钟、久坐提醒、睡觉状态（久未互动）
- 经验等级、成长页徽章与最近互动记录
- 聊天等行为写入 `interaction-log.json`

## 设置与数据

- 声音效果（点击、开心、提醒、升级轻提示音）
- 导出本地数据：`schemaVersion: 2`，含设置、宠物、聊天、互动日志等
- 界面语言：当前仅中文；多语言将单独开发
- 云同步：**规划中**；宠物 `userId` 字段已预留，请用导出备份换机

## 账号

- **须登录**：注册/登录后使用（已移除离线模式）
- 配置后台 API 地址后走真实后台（设置或 `.env`）
- 支付与云同步尚未接入
