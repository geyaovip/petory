# B1.2 交付说明

> 版本：**B1.2.0** · 代码目录：`server/`

## 新增能力

| 模块 | API | 说明 |
|------|-----|------|
| Kimi 代理 | `POST /api/chat/send` | 服务端调用 Moonshot，Key 仅存 `server/.env` |
| 对话额度 | `GET /api/chat/quota` | 免费 20 / Pro 9999 次/日 |
| 用户概览 | `GET /api/me` | 增加 `chatQuota` 字段 |
| 对话日志 | DB `ChatLog` | 仅存 metadata（字数、耗时、状态），**不存全文** |
| 管理后台 | `/admin/` | 今日对话统计、对话日志列表 |
| 风控 | 内置 | 用户 10 次/分钟、IP 30 次/分钟 |

## 请求格式

```json
POST /api/chat/send
{
  "message": "你好呀",
  "history": [
    { "id": "...", "petId": "...", "role": "user", "content": "...", "createdAt": "..." }
  ],
  "pet": {
    "petId": "optional-uuid",
    "name": "咪咪",
    "personality": "温柔陪伴型",
    "userCallName": "主人"
  }
}
```

成功响应：

```json
{
  "success": true,
  "message": { "role": "assistant", "content": "..." },
  "bubbleText": "...",
  "chatQuota": { "dailyLimit": 20, "usedToday": 1, "remainingToday": 19 }
}
```

## 环境变量

```env
KIMI_API_KEY=...
KIMI_API_BASE=https://api.moonshot.cn/v1
KIMI_MODEL=moonshot-v1-8k
```

## 迁移

```bash
cd server
npm run db:up
npm run db:push
npm run dev
```

数据库配置见 [DATABASE.md](../../backend/DATABASE.md)。
