# B1.1 交付说明

> 版本：**B1.1.0** · 代码目录：`server/`

## 新增能力

| 模块 | API | 说明 |
|------|-----|------|
| 多姿势 batch | `POST /api/generation/batch` | 一次请求生成 N 个 pose，**扣 1 次**主额度 |
| 姿势补全 | `POST /api/generation/complete-poses` | `poses` JSON 数组，**不扣**主额度 |
| 单姿势重生成 | `POST /api/generation/regenerate-pose` | **不扣**主额度 |
| 批次查询 | `GET /api/generation/batch/:id` | 含各 pose 子任务状态 |
| 可用姿势 | `GET /api/generation/poses` | 按 plan 返回 3/6 种 |
| 兑换码核销 | `POST /api/redeem` | `{ code }` |
| 权益校验 | 生成前 | Pro 专属风格、姿势按 plan 校验 |
| 管理后台 | `/admin/` | 兑换码 CRUD、生成批次列表 |

## 额度规则（与客户端一致）

| jobType | 扣 dailyGenerationLimit |
|---------|-------------------------|
| `full_batch` | 待机 `idle` 成功扣 **1** |
| `pose_completion` | 不扣 |
| `single_pose_regen` | 不扣 |

## 默认兑换码（首次启动种子）

- `PETORY-PRO-DEMO`（9999 次）
- `PETORY-PRO-TEST`（100 次）

## 迁移

```bash
cd server
npm run db:up
npm run db:push
npm run dev
```

## 验收示例

```bash
# 兑换 Pro
curl -s -X POST http://localhost:8787/api/redeem \
  -H "Authorization: Bearer <token>" \
  -H 'Content-Type: application/json' \
  -d '{"code":"PETORY-PRO-DEMO"}'

# 多姿势 batch（Pro 用户，poses 可省略则按 plan 默认）
curl -s -X POST http://localhost:8787/api/generation/batch \
  -H "Authorization: Bearer <token>" \
  -F "image=@test.jpg" \
  -F "styleType=petory"
```
