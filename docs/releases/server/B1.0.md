# B1.0 交付说明

> 版本：**B1.0.0** · 代码目录：`server/`

## 已实现

| 模块 | 状态 | 说明 |
|------|------|------|
| 假登录 | ✅ | `POST /api/auth/register`、`/login`（邮箱 + 密码） |
| JWT | ✅ | Bearer token，7 天有效 |
| 用户 | ✅ | `GET /api/me`、用户落库 PostgreSQL |
| 设备 | ✅ | `POST /api/devices/register` |
| 生成额度 | ✅ | 免费 3 / Pro 50（与 `PLAN_LIMITS` 一致），日切重置 |
| 生成任务 | ✅ | `POST /api/generation/jobs`（单 pose）、`GET .../jobs/:id` |
| MiniMax 代理 | ✅ | Key 仅 `server/.env` |
| 本地存储 | ✅ | `server/uploads/` + `/uploads/*` 静态访问 |
| 风控 | ✅ | 用户 3/min、IP 20/min、10MB 上限 |
| 管理 API | ✅ | 用户/任务/额度/Pro/禁用 |
| 管理后台 UI | ✅ | `/admin/` 静态页 |
| 管理员种子 | ✅ | 首次启动创建 `ADMIN_EMAIL` |

## 本地启动

```bash
npm run server:install
cd server && cp .env.example .env
# 配置 MINIMAX_API_KEY、DATABASE_URL（见 DATABASE.md）
npm run db:up
npm run db:push
npm run server:dev
```

## 验收自检

```bash
# 注册
curl -s -X POST http://localhost:8787/api/auth/register \
  -H 'Content-Type: application/json' \
  -d '{"email":"test@petory.app","password":"123456"}'

# 登录拿 token 后查额度
curl -s http://localhost:8787/api/generation/quota \
  -H "Authorization: Bearer <token>"
```
