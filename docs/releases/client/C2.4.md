# C2.4 阶段 B 体验增强

> 版本：**2.2.0** · 阶段 B 高优先级三项

## 已实现

| 项 | 说明 |
|----|------|
| 维护公告全局展示 | 除登录/设置外，聊天、引导、桌宠管理等窗口顶部展示 `maintenanceNotice`（`AppShell`） |
| 生成进度分阶段 | 上传 → 云端生成 → 本地下载抠图；`GeneratingPage` 按 `phase` 切换文案与进度条 |
| 失败重试 | 远程 API 与图片下载自动重试（最多 3 次）；错误页区分网络/限流/维护/登录过期 |
| 401 渲染层提示 | `auth.sessionExpired` IPC；聊天页与全局顶栏展示「登录已过期」 |

## 生成错误码

| code | 场景 |
|------|------|
| `network_error` | 无法连接后台或下载失败 |
| `rate_limit` | HTTP 429 |
| `service_disabled` | 生成服务维护 |
| `auth_expired` | Token 过期（401） |

## IPC

| 通道 | 说明 |
|------|------|
| `auth.sessionExpired` | 主进程检测到 401 时广播，`{ message }` |
| `pet.generationProgress` | 新增可选字段 `phase`: `upload` \| `remote` \| `local` |
