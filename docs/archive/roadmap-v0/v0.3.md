# V0.3 — AI 对话版

## 版本目标

证明桌宠有陪伴感，能以宠物身份与用户对话，而不是通用 AI 助手。

## 功能范围

### 必须做

1. 聊天面板（气泡点击 / 右键「和它说话」/ 快捷键）
2. 接入 Kimi（Moonshot）对话 API
3. 宠物人格系统提示词（名称 + 性格）
4. 回复简短，适合气泡展示
5. 可选：最新回复摘要同步到桌宠气泡
6. 聊天记录本地保存（可开关）
7. 应用启动时问候气泡

### 不做

- 语音聊天 / TTS
- 工具调用 / 办公自动化
- 番茄钟、久坐提醒（V0.4）
- 成长系统（V0.4）
- 多风格生成

## 人格选项

温柔陪伴型 · 元气鼓励型 · 傲娇吐槽型 · 安静治愈型 · 严格监督型

## 系统提示词原则

- 以宠物身份回复，不像正式助手
- 短句、自然、有陪伴感
- 不提供医疗、法律、投资等高风险建议

## 开发任务

| # | 任务 |
|---|------|
| 1 | 聊天面板 UI（输入框 + 消息列表） |
| 2 | Kimi 对话 API 封装（主进程，OpenAI 兼容接口） |
| 3 | 人格 prompt 模板注入 |
| 4 | 聊天气泡联动（点击桌宠 → 气泡 → 打开面板） |
| 5 | InteractionLog 记录 |
| 6 | `enableChatHistory` 设置项 |
| 7 | 启动问候语逻辑 |

## 验收标准

- [ ] 点击气泡或右键可打开聊天面板
- [ ] AI 按宠物名称和性格回复
- [ ] 回复风格简短、有陪伴感
- [ ] 聊天记录可本地保存并在重启后读取
- [ ] 关闭聊天历史后不再持久化新记录
- [ ] 启动时桌宠有问候气泡
