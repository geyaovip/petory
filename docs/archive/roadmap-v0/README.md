# Petory 早期产品路线归档（v0.x-v1.x）

> 历史归档：记录项目早期设想，仅用于追溯。当前功能与计划请看 [产品文档](../../product/FEATURES.md) 和 [Backlog](../../product/BACKLOG.md)。

各版本独立说明，开发时按顺序推进，不要跳版。

## 版本总览

| 版本 | 目标 | 文档 |
|------|------|------|
| **V0.1** | 证明桌宠能在桌面运行 | [v0.1.md](./v0.1.md) |
| **V0.2** | 用户可上传图片并 AI 生成专属桌宠 | [v0.2.md](./v0.2.md) |
| **V0.3** | 桌宠有 AI 陪伴感（对话） | [v0.3.md](./v0.3.md) |
| **V0.4** | 形成专注 + 成长闭环 | [v0.4.md](./v0.4.md) |
| **V0.5** | 双平台可测试、设置与错误处理完善 | [v0.5.md](./v0.5.md) |
| **V1.0** | 可公开发布 | [v1.0.md](./v1.0.md) |
| **V1.1+** | 多风格、多姿势等增强能力 | [v1.1-plus.md](./v1.1-plus.md) |

## 依赖关系

```
V0.1 → V0.2 → V0.3 → V0.4 → V0.5 → V1.0 → V1.1+
```

## 平台策略

- V0.1–V0.4：单平台（macOS 或 Windows，开发者主环境优先）
- V0.5：开始双平台打包与适配
- V1.0：macOS + Windows 正式安装包
