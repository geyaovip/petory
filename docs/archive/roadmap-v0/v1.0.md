# V1.0 — 正式版

## 版本目标

可公开发布的产品版本：官网下载、自动更新、基础账号与商业化能力就绪。

## 功能范围

### 必须做

**分发与运维**

1. 官网 + 下载页
2. 自动更新（electron-updater）
3. 崩溃日志 / 基础遥测（可选，需隐私说明）
4. 用户协议 + 隐私政策

**账号体系（基础）**

1. 注册 / 登录
2. 宠物数据结构与账号关联预留
3. 本地模式仍可离线使用（降级策略）

**商业化（基础）**

1. 免费版：1 只宠物、Petory Style、有限 AI 对话额度
2. Pro 版能力规划落地（至少部分）：多宠物、更高对话额度等
3. 生成次数限制或兑换码机制
4. 支付或兑换码（按实际接入方案）

**产品完善**

1. 示例宠物（「先体验示例宠物」入口）
2. 新用户引导打磨
3. 性能与启动速度优化
4. 内容安全提示强化

### 不做（留给 V1.1+）

- 多风格选择器（见 [v1.1-plus.md](./v1.1-plus.md)）
- 多姿势状态图
- Live2D / 语音
- 社区分享
- 云端全量同步（可部分实现）

## 验收标准

- [ ] 用户可从官网下载并安装（macOS + Windows）
- [ ] 应用可检查并安装更新
- [ ] 新用户可完成引导并创建桌宠
- [ ] 账号登录可用（或明确标注 beta）
- [ ] 免费版额度限制生效
- [ ] Pro / 兑换码可解锁对应能力
- [ ] 隐私政策与用户协议可访问
- [ ] V0.1–V0.5 全部核心验收项仍通过

## 免费版 vs Pro（参考）

| 能力 | 免费版 | Pro |
|------|--------|-----|
| 宠物数量 | 1 | 多只 |
| 风格 | Petory Style | + 后续扩展风格 |
| AI 对话 | 每日有限 | 更高 / 无限 |
| 云端同步 | 无 | 有（若实现） |
