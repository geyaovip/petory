# V0.4 — 轻效率与成长闭环

## 版本目标

把专注、提醒、互动行为与桌宠成长绑定，形成「使用 → 反馈 → 再来」的闭环。

## 功能范围

### 必须做

**番茄钟**

1. 默认 25 分钟专注 / 5 分钟休息（可配置）
2. 开始 / 暂停 / 结束
3. 专注中桌宠 `focus` 动画（呼吸）
4. 完成后 +10 经验、`happy` 动画、鼓励气泡
5. 记录 FocusSession

**久坐提醒**

1. 默认每 60 分钟提醒（30/45/60/90 可选）
2. 到时间桌宠 `remind` 动画 + 气泡
3. 用户确认休息后重置计时
4. 连续跳过可触发 `angry` 动画（短暂）

**成长系统**

1. 经验值来源：番茄钟 +10、每日首次打开 +5、聊天 +2、连续使用 +5 等
2. 等级 Lv1–Lv5
3. 成长页：名称、等级、经验条、今日专注次数、连续天数、最近互动

**状态表现**

- MVP 仅用 CSS 动画 + 气泡，不生成多姿势图片

### 不做

- 装饰 / 动作解锁（后续版本）
- 多宠物
- 云端同步
- 语音提醒（可选音效可简单实现）

- [ ] 重启后等级、经验、设置不丢失
- [ ] 各状态动画在单张 `pet.png` 上正常表现
