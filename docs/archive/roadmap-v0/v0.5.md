# V0.5 — 双平台测试版

## 版本目标

产出可给外部测试的安装包，完善设置、错误处理与数据管理，开始 macOS + Windows 双平台适配。

## 功能范围

### 必须做

**打包发布**

1. macOS `.dmg` 安装包
2. Windows `.exe` / NSIS 安装包
3. 双平台透明窗口、拖拽、置顶验证

**设置页**

1. 开机自启动
2. 桌宠置顶开关
3. 桌宠大小（小/中/大）
4. 桌宠透明度
5. 声音开关
6. 久坐 / 番茄钟提醒开关
7. MiniMax API Key 配置
8. 宠物性格调整
9. 退出应用

**数据与隐私**

1. 清除聊天记录
2. 删除宠物图片
3. 删除全部本地数据
4. 导出本地数据

**体验完善**

1. 统一错误提示与空状态
2. 用户反馈入口（邮件 / 表单 / GitHub Issue 链接均可）
3. 宠物管理页（单宠物 MVP，列表结构预留多宠物）

### 不做

- 自动更新（V1.0）
- 账号登录（V1.0）
- 官网（V1.0）
- 付费 / 订阅（V1.0）
- 点击穿透（后续版本）

- [ ] macOS 安装包可安装运行
