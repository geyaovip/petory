下载 Petory

选择你的系统，安装后即可创建桌宠，或先体验示例宠物。

当前版本 —

macOS

支持 Apple Silicon 与 Intel（Universal），需要 macOS 12 或更高版本。

下载 macOS 版

Universal · macOS 12+

Windows

适用于 Windows 10 / 11（64 位）的安装包。

下载 Windows 版

64 位 · Windows 10 / 11

系统要求

macOS

  • macOS 12 Monterey 或更高版本
  • Apple Silicon 或 Intel（Universal 安装包）
  • 约 200 MB 可用磁盘空间

Windows

  • Windows 10 / 11，64 位系统
  • 约 150 MB 可用磁盘空间
  • 建议开启显卡硬件加速以获得更流畅动画

通用

  • 登录账号、生成桌宠与 AI 对话需要网络连接
  • 首次使用可先体验示例宠物，无需上传照片
  • 在应用「设置 → 更新」中可检查新版本

安装后

  • 首次打开可体验示例宠物，或上传自己的照片创建桌宠
  • 登录账号后可同步每日生成与对话额度
  • 右键点击桌宠可打开聊天、专注与更多功能
